# JK Adhesives - Corporate Website

A modern, responsive static landing page designed for **JK Adhesives**, a premium B2B adhesive manufacturing company. The website highlights the brand's industrial-strength PVA solutions, core products, and trusted industry partnerships.

## üöÄ Features
- **Hero Image Carousel:** Auto-rotating image slider with manual navigation dots to capture user attention.
- **Scroll Animations:** Smooth fade-in effects applied to sections and product cards as they enter the viewport, using the `IntersectionObserver` API.
- **Responsive Design:** Fully responsive layout with a mobile-friendly hamburger menu, ensuring the site looks great on desktops, tablets, and smartphones.
- **Product Showcase:** Clean grid presentation of core products like JK WoodBond Pro, PackSecure, and BindFlex.
- **SEO Ready:** Implements structured JSON-LD schema markup for Organization, Product, and Review types to boost local and product search engine visibility.
- **Smooth Navigation:** Anchor links with smooth scrolling enabled for an enhanced user experience.
